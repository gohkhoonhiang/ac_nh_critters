FAQ
 What is this site about?

When I first started playing AC:NH, I found myself frequently checking the Fandom fish list at different times of the day to find out what kind of fishes I could get and sell at a good price.

The Fandom page was great, but it lacked certain filtering controls, eg. the current hour's list of fishes.

At first, I downloaded the HTML table, dumped the table on Coda.io, and added some filtering options eg. current time, and some grouping options eg. location + shadow size. The downside is that Coda.io loads quite slowly, and sometimes requires sign-in if my session expired.

The loading speed wasn't the problem, the main problem is the limited controls available for more views. For example, I needed to see what would be the fishes that would be gone in the next month, so that I could try to fish those that I've not donated to the museum ASAP. It wasn't trivial to create this view, because of the more complex checking logic on the available months.

Finally, I've had enough and decided to create a simple website like this one to fulfill all my needs.

 Where do you get your data from?

As mentioned in the previous answer, I got my data from the Fandom fish list. The bug list is also from Fandom.

With a Ruby script, I added the list of months and hours based on the raw data's month columns and time range for each critter, and finally combined both northern and southern hemisphere lists into a single JSON format.

 Which browsers are supported?

This site has only been tested on Google Chrome (Version 81.0.4044.129). No guarantee that it will work on other browsers.
